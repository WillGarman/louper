Louper

A zoom circle dev tool. Hold Alt / Option to magnify.

Lightweight

Under 2KB gzipped. No dependencies. Works with any page via a single script tag.

DOM Cloning

Clones the live DOM including CSS state. No canvas rendering, no screenshots — just pure CSS transforms.

Shadow DOM

The magnifier lives in a closed Shadow DOM. Your page styles and the loupe styles never interfere.

60fps

Position updates use GPU-composited CSS transforms batched via requestAnimationFrame.

Configurable

Zoom level, radius, border style, activation key — all customizable via API or data attributes.

Framework Agnostic

Works with React, Vue, Svelte, or plain HTML. Also available as a shadcn component.
